Data Cleaning - Selection

  1. Remove all "unnecessary"" variables
    • Not intended to be used for the analysis
    • Supporting variables - those which only provide aditional information/replacement information about other variables
  2. Remove all "doubt" cases - observations with doubt value is 0
  3. Combine the wounded and killed as "casualties"
  4. Clean all observations with no known location
  5. And at this point, seperate the data set into analysis and visualization
  6. Combine date/month/year into a date object
  7. And finally, rename the variables (because the old variable names are too terrible)

Data Modelling - Model Construction

# Create a model using NaiveBayes
nbc <- NaiveBayes(top_group_name ~ region_name + 
                    attack_type_name + 
                    target_type_name + 
                    weapon_type_name +
                    cas.cat, 
                  data = terror.small, 
                  fL = 1)

pred.nbc = predict(nbc, terror.test)
